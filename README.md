# VK Music Player — неофициальный плеер-обёртка

Это приложение открывает vk.com/audio внутри отдельного окна Windows и добавляет
свой интерфейс поверх: сайдбар, нижнюю панель плеера, полноэкранный "Now Playing"
с крупной обложкой — в духе AMOLED-плееров вроде Dotify.

**Важно понимать, как это устроено:** музыка технически всё ещё грузится и
проигрывается через официальный сайт VK (ты логинишься своим обычным аккаунтом
внутри окна). Мы не трогаем их API и не обходим авторизацию — только меняем
внешний вид и добавляем удобные контролы поверх.

## Шаг 1. Установи Node.js

Скачай и установи LTS-версию с официального сайта: https://nodejs.org
(при установке ничего менять не нужно, просто "Next -> Next -> Install").

## Шаг 2. Открой папку проекта в терминале

Распакуй архив с проектом в любую папку, например `C:\vkmusic-player`.
Затем: зажми Shift и кликни правой кнопкой мыши внутри папки проекта →
"Открыть окно PowerShell здесь" (или "Открыть терминал здесь").

## Шаг 3. Установи зависимости

В открывшемся терминале введи и нажми Enter:

```
npm install
```

Это разово скачает Electron и нужные библиотеки (может занять пару минут).

## Шаг 4. Запусти приложение

```
npm start
```

Откроется окно — залогинься в VK как обычно (сессия сохранится, при следующем
запуске логиниться заново не придётся).

## Шаг 5 (когда всё устраивает). Собери отдельный .exe

```
npm run build
```

Готовый установщик появится в папке `dist`. Его уже можно закинуть на рабочий
стол или в автозагрузку Windows.

## Если "сейчас играет" не отображается

VK меняет разметку своего сайта, и подставленные "на глаз" селекторы в файле
`selectors.js` могут не совпадать с текущей версией сайта. Чтобы поправить:

1. В приложении нажми в сайдбаре "Открыть DevTools VK".
2. В открывшейся панели разработчика нажми на иконку курсора в квадратике
   (обычно верхний левый угол панели) и кликни по названию трека на странице VK.
3. Справа подсветится HTML-элемент — посмотри его класс или атрибуты
   (`data-testid`, `aria-label` и т.д.).
4. Открой `selectors.js` в любом текстовом редакторе (даже в Блокноте) и впиши
   найденный селектор первым в соответствующий список (`trackTitle`,
   `trackArtist`, `cover`, `playPauseButton` и т.д.).
5. Сохрани файл и перезапусти приложение (`npm start` заново, либо просто
   перезапусти собранный .exe).

## Известные ограничения

- Это неофициальное приложение, не связанное с VK. Если VK всерьёз изменит
  сайт, часть функций (определение трека, кнопки) может временно перестать
  работать — само проигрывание при этом не пострадает, потому что оно
  происходит на настоящей странице VK внутри окна.
- Названия разделов в сайдбаре ("Поиск", "Плейлисты" и т.д.) используют
  предполагаемые ссылки. Если какой-то раздел откроет не то, что ожидалось —
  открой нужный раздел вручную внутри окна, посмотри на адрес в DevTools
  (вкладка Network) и поправь `data-url` соответствующей кнопки в `index.html`.
